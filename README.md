Getting Started

These instructions will get you a copy of the project up and running on your local machine.

## Prerequisites

- Node 9.0.0
- Angular/Cli
- MongoDB 

## FrontEnd:
Open a terminal and go to front-end/angular-articles folder and run the next command
nvm use
npm install
npm start

## BackEnd:
Open a terminal and go to back-end folder and run the next command
nvm use
npm install
node app.js

**Open navegador http://localhost:4200/**

**Note** The field story_id it's used as a primary key these means if some history have the same storyid it's going to update these post
